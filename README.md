# Levinson-CSC-240-final-project

Description: for my final project, I am recreating Frogger. There will be multiple rows of cars randomly timed apart moving right to left and the user can move the frog down using a button.

Input hardware: SSD1306 OLED screen, ATTiny416 Xplained, button.

Icons (from character_map.asm):
  Frog = asterisk, 
  Car = block, black
